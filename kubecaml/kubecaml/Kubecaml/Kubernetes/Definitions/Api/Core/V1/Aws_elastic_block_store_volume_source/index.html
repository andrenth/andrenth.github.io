Module Kubecaml.Kubernetes.Definitions.Api.Core.V1.Aws_elastic_block_store_volume_source

Represents a Persistent Disk resource in Aws.

an Aws Ebs disk must exist before mounting to a container. The disk must also be in the same Aws zone as the kubelet. An Aws Ebs disk can only be mounted as read/write once. Aws Ebs volumes support ownership management and Se_linux relabeling.
