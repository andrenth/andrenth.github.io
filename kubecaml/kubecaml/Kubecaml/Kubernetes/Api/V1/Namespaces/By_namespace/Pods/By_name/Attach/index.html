connect Get requests to attach of Pod

  • Parameter tty: Tty if true indicates that a tty will be allocated for the attach call. This is passed through the container runtime so the tty is allocated on the worker node by the container runtime. Defaults to false.
  • Parameter stdout: Stdout if true indicates that stdout is to be redirected for the attach call. Defaults to true.
  • Parameter stdin: Stdin if true, redirects the standard input stream of the pod for this call. Defaults to false.
  • Parameter stderr: Stderr if true indicates that stderr is to be redirected for the attach call. Defaults to true.
  • Parameter namespace: object name and auth scope, such as for teams and projects
  • Parameter name: name of the Pod
  • Parameter container: The container in which to execute the command. Defaults to only container if there is only one container in the pod.
