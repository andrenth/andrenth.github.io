connect Get requests to exec of Pod

  • Parameter tty: Tty if true indicates that a tty will be allocated for the exec call. Defaults to false.
  • Parameter stdout: Redirect the standard output stream of the pod for this call. Defaults to true.
  • Parameter stdin: Redirect the standard input stream of the pod for this call. Defaults to false.
  • Parameter stderr: Redirect the standard error stream of the pod for this call. Defaults to true.
  • Parameter namespace: object name and auth scope, such as for teams and projects
  • Parameter name: name of the Pod
  • Parameter container: Container in which to execute the command. Defaults to only container if there is only one container in the pod.
  • Parameter command: Command is the remote command to execute. argv array. Not executed within a shell.
